  CVD Formula:
  ─────────────
  For each trade in the window:
    if trade.side == "buy":   cvd += trade.amount   (aggressive buy = hit ask)
    if trade.side == "sell":  cvd -= trade.amount   (aggressive sell = hit bid)

  CVD Trend:
    Compare first-half delta vs second-half delta of lookback window
    second_half > first_half * 1.10 → "rising"
    second_half < first_half * 0.90 → "falling"
    else                             → "flat"

  Tool                    What it does                                Priority

  ATR Position Sizing     Size position so max loss = 1.5% capital    MUST-HAVE
                          qty = risk_usd / |entry - stop_loss|
  Kelly Criterion         Optimal bet size based on win rate + R:R    HIGH
  (fractional 25%)        Use 25% of full Kelly for safety
  Daily Loss Limit        Halt after 3% daily drawdown                MUST-HAVE
  Consecutive Loss        Halt after 5 losses in a row                MUST-HAVE
  Counter
  Max Drawdown Tracker    Max peak-to-trough equity decline           MUST-HAVE

                          Target: > 1.5
  Sharpe Ratio            Risk-adjusted return metric                 HIGH
                          Target: > 1.5 weekly
  Sortino Ratio           Like Sharpe but only penalizes              MEDIUM
                          downside volatility
  VaR                     Max expected loss at 95% confidence         MEDIUM
  (Value at Risk)
  Pair Correlation        Correlation between BTC, ETH, SOL, BNB      HIGH
  Matrix                  Avoid doubling up on correlated trades

  Position Sizing Formula:
  ─────────────────────────
    risk_usd       = capital × 0.015         (1.5% of capital at risk)
    price_risk     = |entry_price - stop_loss_price|
    quantity       = risk_usd / price_risk
    position_usd   = quantity × entry_price
    leverage_used  = position_usd / capital
    max_notional   = capital × leverage      (cap at this)

  Kelly Criterion Formula:
  ─────────────────────────
    kelly_pct = (W × R - L) / R
    where:
      W = win rate (e.g. 0.65)
      L = 1 - W   (e.g. 0.35)
      R = avg win / avg loss (e.g. 1.5)
    Example: (0.65 × 1.5 - 0.35) / 1.5 = 0.417 = 41.7% (use 25% = 10.4%)

  Circuit Breakers (auto-halt conditions):
  ──────────────────────────────────────────
    1. Daily loss    >= 3% of starting capital
    2. Consecutive   >= 5 losses in a row
    3. Account drops <= 70% of starting capital
    4. ATR elevated  >= 2× its 20-period average (news event)
    5. Spread        >  5 basis points

SECTION 14 — THE 7-SIGNAL CONFLUENCE SCORING SYSTEM
════════════════════════════════════════════════════════════════════════════════

  Each signal scores +1 (bullish), -1 (bearish), or 0 (neutral).
  Total score range: -7 to +7.
  Trade only when |score| >= 4.

  Signal    Indicator           Bullish (+1)              Bearish (-1)
  ────────  ──────────────────  ────────────────────────  ────────────────────────
  1         EMA trend (9/21)    EMA9 > EMA21              EMA9 < EMA21
  2         RSI (14)            RSI <= 35 (oversold)      RSI >= 65 (overbought)
  3         Bollinger Band      Price at lower band       Price at upper band
  4         VWAP deviation      Price < VWAP by 0.10%+    Price > VWAP by 0.10%+
  5         OBI                 OBI >= +0.40              OBI <= -0.40
  6         CVD trend           Trend = "rising"          Trend = "falling"
  7         Volume              Vol >= 1.2x 20-avg        Vol >= 1.2x 20-avg
                                AND price < VWAP          AND price > VWAP

  Decision Logic:
  ────────────────
    Score >= +4  →  LONG  bias  →  call Claude AI
    Score <= -4  →  SHORT bias  →  call Claude AI
    -4 < score < +4  →  NEUTRAL  →  skip (no AI call)

  Additional Hard Gates (must ALL pass before calling AI):
  ──────────────────────────────────────────────────────────
    spread_bps     <= 5.0      (not too wide to enter)
    atr_elevated   == False    (not in news event)
    open_positions < 2         (not at max capacity)
    daily_trades   < 30        (not overtraded today)
    halted         == False    (circuit breakers clear)

  ───────────────────────────────────────
